# Get Started

This is a normal page, which contains VuePress basics.

# Pages

You can add markdown files in your vuepress directory, every markdown file will be converted to a page in your site.

See routing (opens new window) for more details.

# Content

Every markdown file will be rendered to HTML, then converted to a Vue SFC (opens new window).

VuePress support basic markdown syntax and some extensions (opens new window), you can also use Vue features (opens new window) in it.

# Configuration

VuePress use a .vuepress/config.js(or .ts) file as site configuration (opens new window), you can use it to config your site.

For client side configuration (opens new window), you can create .vuepress/client.js(or .ts).

Meanwhile, you can also add configuration per page with frontmatter (opens new window).

# Layouts and customization

Here are common configuration controlling layout of @vuepress/theme-default:

  • navbar (opens new window)
  • sidebar (opens new window)

Check default theme docs (opens new window) for full reference.

You can add extra style (opens new window) with .vuepress/styles/index.scss file.
